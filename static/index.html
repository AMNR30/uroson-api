Portal Data Pasien

Buat / Ambil data pasien

Data yang disimpan di sini akan langsung terbaca oleh perangkat UROSON di ruang pemeriksaan. Kamu juga bisa mengambil data pasien dari server API mana pun (asal formatnya sama).

Identitas Pasien
ID ini yang nanti dimasukkan/discan di UROSON untuk memanggil data pasien. Klik "Ambil dari Server" untuk mengisi form otomatis dari data yang sudah ada.
Terisi otomatis dengan waktu saat ini — silakan ubah manual jika perlu (tanggal & jam).
Fasilitas & Tenaga Medis
Pengaturan Server (API)
Kosongkan untuk memakai server yang sama dengan halaman ini. Dipakai untuk mengambil (GET) maupun menyimpan (POST) data pasien — server tujuan cukup mengikuti format kontrak {"status": true/false, "data": {...}}. Kunci akses (API key) terkirim otomatis di balik layar, tidak perlu diisi manual. Alamat API tersimpan otomatis di browser.
Tombol "Ambil dari Server" memanggil GET /api/patients/{patientid} ke Alamat API di atas dan mengisi form otomatis jika data ditemukan. Tombol "Simpan ke Server" menulis ke POST /api/patients — form ini tidak pernah menghapus data lama.
